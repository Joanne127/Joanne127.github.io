第一次用LeTaX的经历是让我终生难忘的。参加了19年1月的美赛，临近最后提交时间我都不断地在做着排版上的调整，机房的椅子没有靠背，从前一天傍晚肝到第二天早上，腰酸背痛，不知疲倦。所以想借此机会整理一下，指不定哪天会再用上。

下载与安装

下载地址

Attention

耗时比较久（软件比较大），安装时以管理员身份运行。

检查是否配置系统环境变量。控制面板-系统和安全-系统-高级系统设置-环境变量-系统变量-Path。

环境变量 是什么呢？其实我们可以把它理解为【系统的视线范围】，没错，配置进入了环境变量的程序，就等于是进入了系统的视线范围，打开DOS命令窗口后输入程序名，系统就会把在其视线内的（环境变量内）的程序找出来，如果程序没有配置进入环境的变量的话，那系统自然就找不到。（摘自 https://jingyan.baidu.com/article/a17d5285c9b0c48099c8f26a.html）


博主用的是 TeXLive + Atom ，TeXLive自带编辑器TexWorks editor，不过我更喜欢Atom，通过安装以下几个包：

  • language-latex
  • latex
  • pdf-view

可以实现舒适编辑和即时预览。


在tex文件下进行编辑， ctrl + alt + B 来生成/更新PDF文档，或 Packages-LaTeX-Build 也可以达到同样效果。
